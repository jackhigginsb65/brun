1; brun
A simple tool to compile and run C/C++ files quickly

2; Build and installation
make -> sudo make install -> make clean

    2.1; To build, run the command 'make'. This will create the compiled binary
    2.2; To install, run the command 'sudo make install' to install the compiled binary to your /usr/local/bin/
    2.3; Run the command 'make clean' to delete the original compiled binary in your brun directory
