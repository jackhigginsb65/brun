# brun
A simple tool to compile and run C++ files quickly.

## Build and installation
To build, run the command 'make'. This will create the compiled binary
Followed my 'sudo make install' to install the compiled binary to your /usr/local/bin/
Finally, run the command 'make clean' to delete the original compiled binary
